The RWDUHJ TIRM contains the following folders:

1. Coordinator_Guide-PDFs
2. Datafiles For Faculty	
3. Datafiles For Students
4. Milestone_for_Faculty
5. Milestone_for_Students
6. Slides-PPS
7. Session Plan
8. Implementation Manual

The Coordinator_Guide-PDFs folder contains the coordinator guide (.pdfs) for the RWDUHJ course.

The Datafiles For Faculty folder contains the input files and solutions for the activities and exercises given in chapters. In addition, it also contains the animations and intro animation.

The Datafiles For Students folder contains input files required to perform the MR exercises.

The Milestone_for_Faculty folder contains the milestone for the faculty.

The Milestone_for_Students folder contains the milestone for the students.

The Slides-PPS folder contains the session slides for the faculty to conduct the sessions. 

The Session Plan folder contains the session plan according to which the sessions need to be conducted.

The Implementation Manual folder contains the steps for installing required tool to perform activities and exercises given in the course.

The file names of the coordinator guide have been given in the format aa_bb_cc_dd where in 

aa denotes the print order
bb denotes the course name
cc denotes the deliverable 
dd denotes the chapter number

The order of printing should be the increasing order of the print order - that is aa.
