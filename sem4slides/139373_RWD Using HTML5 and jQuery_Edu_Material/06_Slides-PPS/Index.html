Website Layouts

Fixed-width Layout

In the fixed-width layouts, the width of a website is fixed by using a certain number of pixels, preferably the measure is 960 pixels. The 960 pixels are best suited for the grid structure because it is easily divisible by 3, 4, 5, 6, 8, 10, 12, and 15. The width of the Web page remains the same, regardless of the browser window in which the Web page is accessed.

Fluid or Liquid Layout

The fixed-width layouts usually do not work well when the screen size of a device is less than 960 pixels. This is because the website does not get fully displayed. Problems also arise when the screen size of a device is greater than 960 pixels as website layouts usually get surrounded by white space, which makes the website look unappealing. This is one of the oldest Web designing which was used where the screen size was fixed and there were limited devices.

Elastic Layout

The elastic layout is similar to the fluid layout to some extent. It only differs in the unit of measurement used for the layout. The unit of measurement for the elastic layout is neither pixels nor percentages. Here, the unit of measurement is em. An em is defined in the font-size CSS rule. 1em is equal to the current font size. Similarly, 2em means two times the size of the current font. For example, if the element is displayed with the font size of 12pt then, 2em will be equal to 24pt.

Hybrid Layout

The hybrid layout is a combination of two or more layouts. For example, a Web Designer can use pixels to define certain elements, such as sidebar or footer, and can use ems or percentages for the main content that is to be displayed on the website.
